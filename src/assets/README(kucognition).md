# Nail Scanning for Systemic Disease Detection 🩺

This project is a **machine learning model for classifying nail conditions** that may be associated with **systemic diseases**, including cardiovascular and other internal health conditions.

The model analyzes nail images and classifies them into **8 different nail conditions**, which can act as **visual indicators of underlying health issues**.

> ⚠️ This project is for research and educational purposes only and is **not a medical diagnostic tool**.

---

## 🔍 Overview

Nails can reflect changes in the body caused by systemic diseases such as:
- Heart disease
- Circulatory disorders
- Nutritional deficiencies
- Chronic illnesses

This project aims to:
- Use **computer vision and deep learning**
- Detect and classify **8 nail conditions**
- Support early research into **non-invasive health screening**

The model was **trained on approximately 16,000 labeled nail images**, providing a diverse dataset for robust classification performance.

---

## 🧠 Nail Conditions Classified

The model classifies the following **8 nail-related conditions**:

1. Clubbing  
2. Beau’s Lines  
3. Terry’s Nails  
4. Koilonychia (Spoon Nails)  
5. Leukonychia  
6. Onycholysis  
7. Melanonychia  
8. Normal / Healthy Nails  

(Some of these conditions are commonly linked to systemic or cardiovascular diseases.)

---

## ⚙️ Model Details

- **Approach:** Deep Learning (CNN-based)
- **Framework:** Python, TensorFlow / Keras
- **Training Data:** ~16,000 nail images
- **Input:** Nail images
- **Output:** One of 8 nail condition classes
- **Training Notebook:** `NAIL_DISEASE_11_TRAINING.ipynb`
