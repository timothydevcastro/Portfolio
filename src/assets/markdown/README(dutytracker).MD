# Duty Hours Calculator

A web-based duty hours tracking system built for **Financial Aid Scholars of De La Salle University–Dasmariñas**.

The application allows scholars to encode duty sessions, track required hours, add event credits, and generate printable reports based on official duty rules.

---

## Features

- Encode duty entries with automatic hour computation
- Configurable required hours tracking
- Real-time progress summary and remaining hours
- Event credit management (seminars, assemblies, etc.)
- Printable duty reports
- CSV export for record submission
- Rule-based validation for duty schedules

---

## Tech Stack

- HTML5  
- CSS3  
- JavaScript (Vanilla)  
- GitHub Pages (Deployment)

---

## Deployment

This project is deployed using **GitHub Pages** and runs entirely on the client side.

---

## Notes

- Data is stored locally using `localStorage`
- No backend or database is required
- Designed specifically for DLSU-D Financial Aid Scholars
